# VerifiedSkid

**VerifiedSkid** is a huge, chaotic collection of tools, games, experiments, and random pages — all in one retro-styled website. Built in classic 2000s HTML style with minimal CSS (basically none).

# Live Site
- Hosted on GitHub Pages: https://verifiedskid.github.io/

# Notes
- No accounts, no tracking, no ads — just pure chaotic content.
- Best viewed in any browser, old-school or modern.
